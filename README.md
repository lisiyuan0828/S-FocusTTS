## 产品简介

**S-FocusTTS** 是一款常驻后台的桌面语音输入工具。

在任意应用（浏览器、IDE、IM、Office、终端……）中按下全局快捷键，屏幕中央浮出一颗呼吸态小球开始录音；再次按下快捷键，系统将语音本地转写为文字，并**自动注入到你原本聚焦的输入框**——就像你亲手敲出来的一样。

> **一句话价值**：按下即说，松手成字，焦点不丢，全程离线。

## 核心功能

### 🎙️ 全局快捷键唤起
- 默认 `Cmd/Ctrl + Shift + Space`，任意前台应用下均可响应
- 支持自定义快捷键，注册冲突时给出明确提示
- 一键启停：按一次开录，再按一次转写并注入

### 🔮 悬浮呼吸球 UI
- 出现在**鼠标所在显示器**的中央，多屏自动跟随
- 无边框 / 透明背景 / 永远置顶 / 鼠标穿透 / **不抢焦点**
- 音量驱动的实时呼吸缩放，转写中切换为加载态，异常态闪红提示
- 紧凑模式下为纯色科技感小球，状态差异通过主色与辉光表达

### 🧠 本地语音转写（ASR）
- 引擎：[whisper.cpp](https://github.com/ggerganov/whisper.cpp)，完全离线、零网络依赖
- 默认模型 `ggml-medium-q5_0`，中文准确率 ≥ 90%
- 支持 `tiny / base / small / medium / large-v3-turbo` 任意切换与按需下载
- 中英文混合识别，可在设置中强制语言
- 内置常见幻觉过滤（"谢谢观看""请订阅"等）与可选自动标点

### ⌨️ 智能文本注入
- **主策略 Keyboard**：模拟 Unicode 按键直接写入当前焦点窗口
- **降级策略 Clipboard**：剪贴板粘贴并自动恢复原内容
- 基于文本长度自动切换（默认阈值 200 字），用户亦可强制指定
- 录音前记录原焦点窗口，注入前精准回切，**确保焦点零丢失**

### 🎧 高保真录音采集
- 16 kHz / 16-bit / 单声道 PCM，贴合 whisper 推荐输入
- 单次最长 5 分钟，超时自动停止并转写
- 首次使用引导用户完成麦克风授权

### 📜 历史记录面板
- 托盘菜单或 `Cmd/Ctrl + Shift + H` 打开
- 最近 100 条转写记录，支持搜索、复制、删除、导出 TXT / Markdown
- 默认**只存文本不存音频**，隐私可控

### ⚙️ 设置中心
- 快捷键自定义
- ASR 模型切换与下载管理
- 注入策略、识别语言、历史开关、开机自启
- 呼吸球外观（尺寸、颜色、位置偏移）

### 🛎️ 系统托盘
- 左键：打开历史记录
- 右键菜单：开始 / 停止录音、历史、设置、关于、退出

### 🔔 版本检查（仅提示，不强推）
- 启动 30 s 后异步检查新版，失败静默
- 发现新版时托盘加小红点并在设置面板顶部提示下载链接
- 支持"忽略此版本"，永不打扰

### 🧭 首次启动引导
- 三屏引导：欢迎 → 权限（麦克风 + 辅助功能）→ 快捷键实测
- 权限未全部就绪不得进入下一步，确保开箱即用

## 产品特性

- **完全离线**：语音与文本永不上传，敏感场景可放心使用
- **不抢焦点**：悬浮窗 `focusable=false`、鼠标穿透，注入目标始终正确
- **跨平台一致**：macOS 12+ / Windows 10+ 体验对齐
- **高性能**：待机 ≤ 300 MB，录音中 ≤ 600 MB，冷启动 ≤ 3 s
- **识别快**：5 s 语音出字 ≤ 2 s，60 s 语音出字 ≤ 15 s（medium 模型 · 主流硬件）
- **可扩展架构**：主 / 预加载 / 渲染三进程清晰分层，便于后续迭代与多项目复用

## 快速开始

### 环境要求
- Node.js ≥ 20
- pnpm（推荐）或 npm

### 安装与运行

```bash
pnpm install
pnpm dev
```

启动后：
1. 应用常驻后台（macOS 不显示 Dock 图标）
2. 按下 `Cmd+Shift+Space`（Windows 为 `Ctrl+Shift+Space`）唤起呼吸球并开始录音
3. 再按一次快捷键 → 停止录音 → 本地转写 → 自动注入到原输入框

### 类型检查与打包

```bash
pnpm typecheck
pnpm build
```
